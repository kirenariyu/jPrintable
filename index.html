Print Responsively

Please take note that you don't need Foundation and LESS to use this plugin. The design of your web's all yours. You'll be only needing are the following:

  1. jQuery
  2. print-responsively.css for @media print
  3. print-responsively.js for config

The cons for this plugin is you need to place the size of the paper and will only be responsive for that size of the paper.

Introduction

Print Responsively aims to render webpage into paper simply, properly and responsively in every browser; eliminating the default footer and header of user's web browser.

Before, there were rules in CSS that we can use for @media print so we could print our desired webpages properly. But, unfortunately, some of them become deprecated. So I made this plugin to help others give other devs and idea or use it (that would be an honor) for their website or web application.

Getting Started

Print Responsively depends on jQuery. So you must have a jQuery first before calling this plugin to your document.

I created a simple website in this repository give you some examples on how this plugin works. But you will be only needing these files to add in your document's head:

  • css/print-responsively.css
  • js/print-responsively.js

Then add this code to your code:

$(document).printResponsively();

Or if there's only a specific content in your document that you want it printed:

$('#container').printResponsively();

Cheat Sheet (CSS)

You may add additional rules depending on the set design. Just take note that sometimes float elements can destroy your design.

If you encountered some display problem, you also need to look carefully the parent element/s because they can affect the look of your page on print and this can really waste a lot of your time. Your partner-in-crime is inspect console in browsers and try experimenting in styles.
